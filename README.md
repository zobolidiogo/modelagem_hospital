# Modelagem de Banco de Dados - Sistema Hospitalar

![Oracle Data Modeler](https://img.shields.io/badge/Oracle%20Data%20Modeler-F80000?style=for-the-badge&logo=oracle&logoColor=white)
![Modelagem de Dados](https://img.shields.io/badge/Modelagem%20de%20Dados-0078D4?style=for-the-badge&logo=database&logoColor=white)

Modelagem lógica de banco de dados para sistema de gestão hospitalar, desenvolvida como exercício acadêmico durante o curso técnico em Ciência de Dados no CEDUP.

---

## Contexto do Projeto

Atividade prática realizada com o objetivo de aplicar conceitos de modelagem de dados e dominar a ferramenta Oracle Data Modeler. O projeto simula um sistema real de gestão hospitalar, abrangendo os principais processos e entidades de uma instituição de saúde.

---

## Objetivo

Projetar uma estrutura de banco de dados normalizada e eficiente para gerenciar:

- **Pacientes** — Cadastro, histórico e alergias
- **Triagem e Consultas** — Atendimento médico e registros clínicos
- **Receitas e Medicamentos** — Prescrição e controle de medicação
- **Procedimentos Cirúrgicos** — Agendamento e execução de cirurgias
- **Recursos Hospitalares** — Salas, enfermeiros e médicos

---

## Conceitos Aplicados

- Modelagem Lógica Relacional
- Normalização de Dados
- Chaves Primárias (PK) e Estrangeiras (FK)
- Relacionamentos 1:1, 1:N e N:N
- Integridade Referencial
- Cardinalidade e Opcionalidade

---

## Tecnologias Utilizadas

- **Oracle Data Modeler** — Ferramenta de modelagem visual
- **Modelagem Relacional** — Estruturação de banco de dados
